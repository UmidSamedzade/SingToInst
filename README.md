# SingToInst

A project to convert singing/vocal inputs to instrumental tracks or synthesizers.

## Setup

This project uses a Python virtual environment.

### Prerequisites

- Python 3.12 or higher

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/UmidSamedzade/SingToInst.git
   cd SingToInst
   ```

2. The virtual environment is located in `.venv`. Activate it using:
   - **Windows (PowerShell)**:
     ```powershell
     .venv\Scripts\Activate.ps1
     ```
   - **Windows (CMD)**:
     ```cmd
     .venv\Scripts\activate.bat
     ```
   - **macOS/Linux**:
     ```bash
     source .venv/bin/activate
     ```

3. Install requirements (once added):
   ```bash
   pip install -r requirements.txt
   ```
